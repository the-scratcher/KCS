A Technical documentation for my design of computer in sebastian lague's digital logic simulator.

Main chips:
CPU: this reads code and sends it to:
INTerpreters which are basically decoders with extra functionality
RAM includes 256 values and a register with the address
Registers A, B, and C, which store 1 value each
ROM stores the program
ALU-3 does adding & subtracting
ALU-0 +1 and -1
NIL always 0
CLU hopefully does conditionalls

I have a list of commands in a excel file and google sheets
